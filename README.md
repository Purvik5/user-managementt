# User Management API

This is a simple user management API project that includes CRUD endpoints for managing users, validation for user data, and middleware for logging requests.

## Features

- CRUD endpoints: GET, POST, PUT, DELETE for users
- Validation of user data
- Logging middleware
- Ready for deployment to GitHub

## How to run

1. Install dependencies: `npm install`
2. Start the server: `npm start`
3. API will be available at `http://localhost:3000`

## Endpoints

- GET /users - Get all userss
- GET /users/:id - Get user by ID
- POST /users - Create a new user
- PUT /users/:id - Update user by ID
- DELETE /users/:id - Delete user by ID
